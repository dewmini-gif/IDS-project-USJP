About the Project

Our project presents a machine-learning-based Real-Time Intrusion Detection System (IDS) for smart home IoT environments. It identifies cyberattacks such as DDoS, Botnets, and malware using Random Forest and Decision Tree algorithms trained on IoT23, CICIDS2017, and KDD-99 datasets.

Key Features

  • Real-time attack detection and alerts
  • High accuracy with low false positives
  • Interactive dashboard & attack history
  • WebSocket-enabled UI for live updates
  • Secure backend using Flask & MongoDB

Datasets Used

  • IoT23: IoT-specific attacks
  • CICIDS2017: Real-world cyber threats
  • KDD-99: Classical network attacks
